## 全体サマリー（ランク5帯 攻略の要点）

### ゴール
**真・時シリーズ（真・時のつるぎ / 聖剣 / 弓 / 杖 / ピッケル 等）** の完成。ストーリークリア後、トレジャーパンサーから真・伝説レシピをもらい、バカデッカーナ大陸ランク5帯の伝説素材で仕上げる。

### 必須伝説素材と入手先

| 素材 | 入手先エリア | チャレンジ |
|---|---|---|
| 伝説ドン・ホーン | クネック山岳 | 伝説のモンスターを求めて |
| 伝説ギンギラダイヤ | カラット砂漠西 | 伝説の鉱床を求めて |
| 伝説の神原木 | ハネビラの森 | 伝説の木を求めて |

### 周回の最効率テンプレ

1. **アイテムドロップUP装備を一式作成**（最大+110%）— マール評判5 → トリップアップリケで各装備を打ち直し。
2. **仲間編成を塔で済ませる**（対象ライフ所持者）。信頼度でスキル解放。
3. **ターゲットエリアをER3以上**にする。ER5が最効率。
4. **マルチ**: 画面上ゲージMAXで伝説チャレンジ確定発生。**ソロ**: 塔でランク切替を繰り返し再抽選。
5. **大陸移動はNG**（伝説チャレンジがリセット）。
6. **採取職は交互周回**: グラスファイト鉱床（ノーズン山地ER5） ⇄ レイトウマグロ島 で採掘＋釣り同時レベリング。

| エリア | 狙い |
|---|---|
| クネック山岳 ER3+ | 伝説ドン・ホーン |
| カラット砂漠西 ER3+ | 伝説ギンギラダイヤ |
| ハネビラの森 ER3+ | 伝説の神原木 |
| ノーズン山地 ER5 | グラスファイト鉱床（古代のオーブ等）＋レイトウマグロ |
| ドロドロ溶岩地帯 ER5 | 溶岩の竜王石 / ガンモエパンサー / マグマグロ（伝説チャレなし） |
| バカデッカーナ拠点 ER5 | トレジャーパンサー（真・伝説レシピ）/ グランキマイラ周回 |
| グランベリー大平原南・西 ER5 | 伝説野菜 / キングバハ・ハ（王様羊毛） |
| ツバサキ谷 ER5 | 伝説ゴッドツリー / マルマル |

### 整合性チェック（3ファイル間の重複・矛盾確認）

- 伝説ドン・ホーン/ギンギラダイヤ/神原木の入手エリアは items.md・monsters.md・gathering.md で一致。
- グラスファイト鉱床（ノーズン山地ER5）とレイトウマグロ島（ノーズン山地ER5 南西部）の位置関係は items.md / gathering.md / monsters.md で整合。
- アイテムドロップUP装備の仕様（最大+110%、マール評判5 → トリップアップリケ）は items.md・monsters.md で一致。
- 伝説チャレンジ発生条件（ER3以上、マルチゲージMAX or ソロ塔ガチャ、大陸移動でリセット）は全ファイルで一致。
- セキネツ山地・ドロドロ溶岩地帯は伝説チャレンジ非発生の旨を monsters.md と gathering.md に明記。
