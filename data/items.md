## 1. ランク5帯の位置づけ（サマリー）

- バカデッカーナ大陸のランク5以上で **金冠採取ポイント** が発生し、伝説素材の入手が現実的になる。
- 伝説チャレンジは **エリアランク3以上** で発生するが、ランク5では雑魚討伐ポイントも跳ね上がるため、ランク5を軸に周回するのが最効率。
- 最終装備「真・時の○○」は **ストーリークリア後** に解禁。トレジャーパンサーから真・伝説レシピをもらい、ランク5帯の伝説素材で仕上げる流れ。

## 4. アイテムドロップUP装備（周回効率化）

| 部位 | 上昇率 |
|---|---|
| 頭 | +10% |
| トップ / ボトムス | 各+10% |
| 腕 / 足 | 各+10% |
| マント / メガネ / アクセサリ | 各+20% |
| 盾（王国兵士のみ） | +20% |

- **合計最大 +110%**（ドレス枠は両枠消費で非効率）。
- 作成手順: マールのショップ評判5 → トリップアップリケレシピ購入 → 鍛冶屋/大工/裁縫師/錬金術師で「打ち直し＋追加素材」スキル習得 → 各装備に付与。
- 伝説素材集めを始める前に一式揃えると周回効率が段違い。

## 5. 伝説素材 効率周回の基本手順

1. **マルチプレイ推奨**: 画面上部のゲージをMAXにすると伝説チャレンジが確定発生。
2. **ソロの場合**: 塔に戻ってエリアランク切替（上げ下げ）を繰り返すと伝説チャレンジが再抽選される（「塔ランク切替裏技」）。
3. **大陸移動はチャレンジリセット**: 仲間編成は塔で済ませ、大陸に降りる前にマップで伝説チャレンジの発生有無を確認。
4. **仲間編成**: 採掘師の伝説素材を狙うならデグダス / シルヴィア / グランツなど採掘師ライフ所持の仲間を同行させ、信頼度でスキル解放。
5. **ドロップUP装備**を全身装備した上でチャレンジに挑む。
